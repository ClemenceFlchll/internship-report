Antenna design and fabrication face numerous challenges. One ideal antenna must be small, low cost and with high performances, which englobes great directivity, fast reactivity, MIMO (multiple input, multiple output, ie making several beams at a time) and an ability to be used in different frequencies.
As for today, different antenna co-exist for different uses. For example we have the Dipole antenna, a simple and common antenna which is omnidirectional and is widely used in radio communication systems, television, and other wireless transmission applications. There's also the Yagi antenna, or rake antenna, which consists of several source points. It's like an antenna array whose elements are fed by mutual induction. If strand spacing and length are optimal, the radiation pattern and electronic gain is that of an array.
